# 📊 V-Sales Store (Mobile Sales Dashboard)

## 📌 Project Overview
This project focuses on analyzing mobile sales data from **V-Sales Store** to uncover key business insights.  
The dashboard helps in understanding sales performance, customer behavior, and regional trends.

---

## 🛠 Tools Used
- **Advanced Excel** → Data cleaning & preprocessing  
- **Power BI** → Dashboard creation & visualization  

---

## 📂 Dataset Details
The dataset includes:
- Product / Mobile Model  
- City  
- Sales Amount  
- Quantity Sold  
- Customer Ratings  
- Payment Methods  
- Purchase Date  

---

## 📊 Dashboard Features
- KPI Cards:
  - Total Sales  
  - Total Quantity Sold  
  - Total Transactions  
  - Average Sales  

- Visualizations:
  - Monthly Sales Trends (Line Chart)  
  - Sales by City (Map / Bar Chart)  
  - Sales by Mobile Model  
  - Customer Ratings Distribution  
  - Payment Method Analysis  

- Filters:
  - Month-wise slicer for dynamic analysis  

---

## 📈 Key Insights
- Identified top-performing mobile models  
- Analyzed sales trends across different months  
- Found high-revenue generating cities  
- Evaluated customer preferences using ratings  
- Compared different payment methods usage  

---

## 🎯 Project Objective
To transform raw sales data into meaningful insights and build an interactive dashboard that supports data-driven decision-making.

---

- Creating interactive dashboards in Power BI
